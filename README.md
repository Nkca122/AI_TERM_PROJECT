# Game solution Automation

The project details the development of bots or “AI Agents” for finding solutions to a game, via observing the game state and maximizing rewards, using the programming language “Python” and the multimedia library “Pygame” to implement core mechanics, sensors and neural networks from the ground-up in order to demonstrate the evolution of autonomous systems.

## Dependencies

This project requires the following environment and libraries

- Python (version >= 3.8.x) and pip (version >= 25.x.x)
- Standard python libraries (math, random)
- Pygame (version 2.6.x)

> NOTE <br/> <i>This project requires a basic understanding of coordinate geometry and the python language</i> <br/>
 
## Installation ![PowerShell](https://img.shields.io/badge/PowerShell-%235391FE.svg?style=for-the-badge&logo=powershell&logoColor=white)

- #### Clone the repository
```(bash)
git clone https://github.com/Nkca122/AI_TERM_PROJECT
```

- Open the project folder

- #### Create a python environmant and install the required dependencies

```(bash)
python3.12 -m venv .venv; .\.venv\Scripts\Activate.ps1; pip install "pygame==2.6.*"
```

- Open the `AI-Term-Project` folder and open the jupyter notebook `5. genetic algorithm.ipynb` and select the recently crrated environment for it's kernel

- In the notebook execute all cells in order to start the simulation
